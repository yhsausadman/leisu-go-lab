# leisu-go-lab

## 项目简介

`leisu-go-lab` 是一个用于归档和发布多个独立 HTML 页面的仓库。这里存放的页面内容不针对特定域名或网站，仅作为通用 HTML 文档的集合与版本管理用途。

项目名称中的 "leisu-go" 为仓库标识，"lab" 表示实验性或归档性质的页面集合。

## 目录说明

```
leisu-go-lab/
├── pages/          # 存放独立 HTML 页面文件
├── assets/         # 页面所需的静态资源（图片、样式、脚本等）
├── docs/           # 项目相关文档或说明
└── README.md       # 本文件
```

- `pages/`：主要页面归档目录，每个 HTML 文件为一个独立页面。
- `assets/`：页面依赖的资源文件，按需组织子目录。
- `docs/`：额外的文档记录，如页面设计说明、更新日志等。

## 页面归档说明

本仓库中的 HTML 页面均为独立文档，彼此之间无直接关联。页面内容可能涉及技术演示、实验性布局、工具页面或其他通用用途。

- 每个页面可独立访问，不依赖仓库内其他文件（除非明确引用 `assets/` 中的资源）。
- 页面文件命名建议使用英文小写，以短横线分隔，如 `example-page.html`。
- 仓库不承诺对页面内容进行长期维护或更新。

## 维护说明

- 本仓库由项目维护者不定期更新，可能新增、修改或删除页面。
- 提交新页面时，请将 HTML 文件放入 `pages/` 目录，并确保引用的资源路径正确。
- 如有需要，可在 `docs/` 目录下添加对应的说明文档。
- 欢迎通过 Issue 或 Pull Request 提出建议或报告问题。
